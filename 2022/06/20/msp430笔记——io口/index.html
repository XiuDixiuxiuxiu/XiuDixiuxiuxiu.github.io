1、io 口类型

  • 可独立编程的 I/O 管脚
  • 输入或输出可任意组合
  • 可独立设置的 P1 和 P2 中断，某些器件可能还有其他中断口 39、独立的输入、输出数据寄存器
  • 可独立配置的上拉或下拉电阻

其中只有 P1 和 P2 有中断能力，所有 P1 口的 I/O 管脚中断都来源于**同一个**中断向量 P1IV，

​ 而所有 P2 口的中断都来源于**不同的单一**中断向量 P2IV。

2、io 口寄存器

PxDIR PxREN PxOUT I/O 口配置
0 0 x 输入
0 1 0 下拉输入
0 1 1 上拉输入
1 x x 输出
