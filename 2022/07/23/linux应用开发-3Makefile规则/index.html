Makefile 作用

makefile 就是对文件分别编译,最后链接在一起,避免全部文件都编译一次

Makefile 规则

目标文件不存在

当"依赖"比"目标"新,执行命令

Makefile 语法

即时变量,延时变量,export

A := xxx #A 的值即刻确定,在定义时即确定
B = xxx #B 的值使用到时才确定

即时就是当前赋值,延时就是全局生效,之前的变量也会跟着改变

通配符: %.o

$@ 表示目标
$< 表示第 1 个依赖文件
$^ 表示所有依赖文件

假象目标: .PHONY

.PHONY: clean 假想 clean 文件,就算存在 clean 文件,也能执行 clean 命令
